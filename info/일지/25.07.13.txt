개념 정리
1. 데이터 평면화
= 중첩된 데이터(ex. 딕셔너리 데이터 구조안에 또 다른 딕셔너리 데이터)를 하나의 딕셔너리 구조로 생성하는 것

2. Random Forest(랜덤 포레스트)
= 예측 알고리즘 종류 중 하나로,
= 의사결정 나무(전문가)를 여러개 만들어서 투표로 결정하는 것

3. Gradient Boosting(그래디언트 부스팅)
= 예측 알고리즘
= 실수를 줄여가며 학습하는 방법 (첫번째 모델이 틀린 부분을 그 다음 모델이 보완)
= 특징 : 가장 정확하나, 시간이 오래 걸림

4. Logistic Regression(로지스틱 회귀)
= 선형적으로 확률을 계산하는 방법
= 각 요소에 가중치를 주어 합산
= 특징 : 해석하기 쉽고 빠르다

5. 백테스트(BackTesting)
= 과거의 데이터로 만든 프로그램을 시뮬레이션 해보는것
= 목적 : 실제 돈을 잃기 전 해당 알고리즘 검증

6. 테스트 스위트(Test Suite)
= 여러가지 테스트를 모아놓은 도구
= 모델의 성능을 다각도로 검증함. 마지막 출시 전 종합 검사 같은 것

7. StandardScaler(표준화 스케일러)
= 데이터의 범위가 다르면 모델이 혼동한다.
= ex) 나이 4, 몸무게 450 이라면 몸무게의 값이 크기 때문에 모델이 해당 데이터를 더 중요한 데이터로 인식한다.
= 따라서 모든 데이터를 평균 0, 표준편차 1로 만들어 공정하게 비교한다.
